Проверьте перед запуском

Опубликованный файл checksums сейчас покрывает desktop-архивы. Checksums для Android следует добавить до stable-релиза.

checksum
curl -L -O https://github.com/veilconnect/PrivateDeploy/releases/download/v2.0.12/checksums.sha256
sha256sum -c checksums.sha256
