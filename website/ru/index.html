PrivateDeploy

Инструмент для развертывания прокси на собственных VPS. Создавайте серверы, разворачивайте несколько протоколов и сразу видьте локальные порты Mixed, HTTP и SOCKS.

Создан для управления собственными узлами.

PrivateDeploy не просто локальный прокси-клиент. Он создает VPS, разворачивает учетные данные, синхронизирует ссылки протоколов и показывает локальные порты.

01

Создание в облаке

Создавайте и удаляйте узлы у поддерживаемых провайдеров или на SSH-доступных хостах.

02

Мультипротокол

Разворачивайте Shadowsocks, Hysteria2, VLESS-Reality и Trojan при поддержке выбранного плана.

03

Видимые порты

Mixed, HTTP и SOCKS видны на странице обзора и копируются напрямую.

04

Desktop, mobile и API

Используйте Wails desktop, Android companion или Go HTTP API.

Панель для ясного контроля состояния.

Desktop-приложение показывает трафик, режим и локальные порты на первом экране; облачная панель хранит порты и ссылки каждого узла.
