Build necessária

Verifique antes de executar

O arquivo de checksums publicado cobre atualmente arquivos desktop. Checksums do Android devem ser adicionados antes de uma versão estável.

checksum
curl -L -O https://github.com/veilconnect/PrivateDeploy/releases/download/v2.0.12/checksums.sha256
sha256sum -c checksums.sha256
