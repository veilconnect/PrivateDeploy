Beta / Preview

PrivateDeploy

Ferramenta para implantar proxies em VPS próprios. Crie servidores, implante vários protocolos e veja as portas locais Mixed, HTTP e SOCKS no desktop.

Baixar preview Guia rápido Modelo de segurança

Feito para gerenciar seus próprios nós.

PrivateDeploy não é apenas um cliente proxy local. Ele cria VPS, implanta credenciais, sincroniza links de protocolo e mostra portas locais.

01

Provisionamento em nuvem

Crie e remova nós em provedores compatíveis ou hosts acessíveis por SSH.

02

Implantação multiprotocolo

Implante Shadowsocks, Hysteria2, VLESS-Reality e Trojan quando o plano permitir.

03

Portas visíveis

Mixed, HTTP e SOCKS aparecem na tela principal e podem ser copiadas diretamente.

04

Desktop, mobile e API

Use o app Wails, o cliente Android ou a API HTTP em Go.

Um painel com clareza operacional.

O app desktop mostra tráfego, modo e portas locais na primeira tela; o painel de nuvem mantém portas e links por nó.
