プラットフォームを選択

デスクトップ用アーカイブは CPU アーキテクチャ別です。Android には小さい arm64 版と大きい universal 版があります。

Windows

多くの最新 PC では amd64 を使います。arm64 は Windows on Arm デバイス向けです。

実行前に検証

公開 checksum ファイルは現在デスクトップアーカイブを対象にしています。安定版前に Android の checksum を追加する必要があります。

checksum
curl -L -O https://github.com/veilconnect/PrivateDeploy/releases/download/v2.0.12/checksums.sha256
sha256sum -c checksums.sha256
