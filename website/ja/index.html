Beta / Preview

PrivateDeploy

自分の VPS にプロキシノードを展開するためのツールです。サーバー作成、複数プロトコル展開、Mixed / HTTP / SOCKS のローカルポート確認を一つにまとめます。

Preview をダウンロード クイックスタート セキュリティモデル

自分のノードを管理するために設計。

PrivateDeploy は単なるローカルプロキシクライアントではありません。VPS 作成、認証情報の展開、プロトコルリンク同期、ローカルポート表示まで扱います。

01

クラウド作成

対応クラウドまたは SSH 接続可能なホストでノードを作成、削除します。

02

複数プロトコル

条件が合えば Shadowsocks、Hysteria2、VLESS-Reality、Trojan を展開します。

03

見えるポート

Mixed、HTTP、SOCKS ポートを概要画面で確認し、そのままコピーできます。

04

デスクトップ、モバイル、API

Wails デスクトップ、Android companion、Go HTTP API を利用できます。

運用状況が一目でわかる画面。

デスクトップアプリは通信量、モード、ローカルポートを最初の画面に表示し、クラウドパネルは各ノードのポートとリンクを保持します。
