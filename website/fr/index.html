Beta / Preview

PrivateDeploy

Outil de deploiement de proxies sur vos propres VPS. Creez un serveur, deployeez plusieurs protocoles et voyez directement les ports locaux Mixed, HTTP et SOCKS.

Telecharger la preview Guide rapide Modele de securite

Concu pour gerer vos propres noeuds.

PrivateDeploy n est pas seulement un client proxy local. Il cree des VPS, deploie les identifiants, synchronise les liens de protocole et affiche les ports locaux.

01

Provisioning cloud

Creez et supprimez des noeuds avec des fournisseurs compatibles ou des hotes accessibles par SSH.

02

Deploiement multiprotocole

Deployeez Shadowsocks, Hysteria2, VLESS-Reality et Trojan quand le plan le permet.

03

Ports visibles

Mixed, HTTP et SOCKS sont visibles dans la vue principale et copiables en un clic.

04

Bureau, mobile et API

Utilisez l app Wails, le compagnon Android ou l API HTTP Go.

Un tableau de bord oriente clarte operationnelle.

L app de bureau affiche trafic, mode et ports locaux des la premiere vue; le panneau cloud garde les ports et liens de chaque noeud.
