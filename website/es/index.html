Beta / Preview

PrivateDeploy

Herramienta para desplegar proxies en VPS propios. Crea servidores, despliega varios protocolos y muestra los puertos locales Mixed, HTTP y SOCKS en el escritorio.

Descargar preview Guía rápida Modelo de seguridad

Diseñado para gestionar tus propios nodos.

PrivateDeploy no es solo un cliente proxy local. Crea VPS, despliega credenciales, sincroniza enlaces de protocolo y muestra los puertos locales en la primera pantalla.

01

Creación en la nube

Crea y elimina nodos con proveedores compatibles o hosts accesibles por SSH.

02

Despliegue multiprotocolo

Despliega Shadowsocks, Hysteria2, VLESS-Reality y Trojan cuando el plan lo permite.

03

Puertos visibles

Mixed, HTTP y SOCKS aparecen en la vista principal y se pueden copiar directamente.

04

Escritorio, móvil y API

Usa la app de escritorio Wails, el cliente Android o la API HTTP en Go.

Una vista clara del estado operativo.

La app de escritorio muestra tráfico, modo de operación y puertos locales en la primera vista; el panel de nube conserva puertos y enlaces de cada nodo.
