Beta / Preview

PrivateDeploy

직접 소유한 VPS에 프록시 노드를 배포하는 도구입니다. 서버 생성, 다중 프로토콜 배포, Mixed / HTTP / SOCKS 로컬 포트 확인을 한 곳에서 처리합니다.

Preview 다운로드 빠른 시작 보안 모델

직접 관리하는 노드를 위해 설계되었습니다.

PrivateDeploy는 단순한 로컬 프록시 클라이언트가 아닙니다. VPS 생성, 자격 증명 배포, 프로토콜 링크 동기화, 로컬 포트 표시를 제공합니다.

01

클라우드 프로비저닝

지원되는 클라우드 또는 SSH 접근 가능한 호스트에서 노드를 만들고 삭제합니다.

02

다중 프로토콜 배포

조건이 맞으면 Shadowsocks, Hysteria2, VLESS-Reality, Trojan을 배포합니다.

03

보이는 로컬 포트

Mixed, HTTP, SOCKS 포트를 개요 화면에서 보고 바로 복사할 수 있습니다.

04

데스크톱, 모바일, API

Wails 데스크톱 앱, Android companion, Go HTTP API를 사용할 수 있습니다.

운영 상태를 명확하게 보여주는 대시보드.

데스크톱 앱은 트래픽, 모드, 로컬 포트를 첫 화면에 표시하고 클라우드 패널은 각 노드의 포트와 링크를 보관합니다.

현재 릴리스 상태.

공개 테스트 가치는 있지만 실제 플랫폼과 클라우드 검증이 끝나기 전까지는 Beta / Preview로 표시해야 합니다.
