PrivateDeploy 다운로드

최신 공개 빌드는 2026년 6월 24일 릴리스된 v2.0.12입니다. 이 바이너리는 preview 빌드로 사용하고 실행 전 checksums를 확인하세요.

릴리스 노트 Checksums

플랫폼 선택

데스크톱 압축 파일은 CPU 아키텍처별로 제공됩니다. Android는 작은 arm64 빌드와 더 큰 universal 빌드가 있습니다.

Windows

대부분의 최신 PC는 amd64를 사용합니다. arm64는 Windows on Arm 장치용입니다.

macOS

Apple Silicon은 arm64, Intel Mac은 amd64를 선택하세요.

PrivateDeploy-darwin-arm64.zip5.7 MB
다운로드
PrivateDeploy-darwin-amd64.zip6.2 MB
다운로드

Linux

현재 Linux 산출물은 amd64 압축 파일입니다. DEB/RPM 패키지는 릴리스 게이트로 남아 있습니다.

PrivateDeploy-linux-amd64.zip6.1 MB
다운로드

Android

최신 Android 휴대폰은 arm64를 사용하세요. Universal APK는 더 크지만 더 많은 기기를 지원합니다.

PrivateDeploy-android-arm64.apk51.3 MB
다운로드
PrivateDeploy-android-universal.apk150.2 MB
다운로드

iOS

iOS는 아직 바로 사용할 수 있는 공개 다운로드가 아닙니다. VPNCore.framework, App Group, Network Extension 권한이 필요합니다.

실행 전 검증

공개 checksum 파일은 현재 데스크톱 압축 파일을 포함합니다. 안정 릴리스 전 Android checksum도 추가해야 합니다.

checksum
curl -L -O https://github.com/veilconnect/PrivateDeploy/releases/download/v2.0.12/checksums.sha256
sha256sum -c checksums.sha256
