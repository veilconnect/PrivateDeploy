Beta / Preview

PrivateDeploy

أداة لنشر عقد بروكسي على خوادم VPS تملكها. أنشئ الخادم، انشر عدة بروتوكولات، واعرض منافذ Mixed و HTTP و SOCKS المحلية مباشرة.

تنزيل النسخة التجريبية البدء السريع نموذج الأمان

مصمم لإدارة عقدك الخاصة.

PrivateDeploy ليس مجرد عميل بروكسي محلي. ينشئ VPS، ينشر بيانات الاعتماد، يزامن روابط البروتوكولات، ويعرض المنافذ المحلية بوضوح.

01

إنشاء سحابي

أنشئ واحذف العقد عبر مزودين مدعومين أو مضيفين يمكن الوصول إليهم عبر SSH.

02

نشر متعدد البروتوكولات

ينشر Shadowsocks و Hysteria2 و VLESS-Reality و Trojan عندما تسمح الخطة بذلك.

03

منافذ مرئية

تظهر منافذ Mixed و HTTP و SOCKS في صفحة النظرة العامة ويمكن نسخها مباشرة.

04

سطح مكتب، جوال، API

استخدم تطبيق Wails، عميل Android companion، أو Go HTTP API.

لوحة تعرض حالة التشغيل بوضوح.

يعرض تطبيق سطح المكتب المرور والوضع والمنافذ المحلية في أول شاشة، ويحفظ قسم السحابة منافذ وروابط كل عقدة.
