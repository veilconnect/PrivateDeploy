Mulai cepat

Deploy node dan temukan port proxy lokal.

Instal app desktop, tambahkan provider, buat node VPS, lalu gunakan endpoint Mixed, HTTP, atau SOCKS lokal.

Instal build preview

Unduh build untuk platform Anda dan verifikasi checksum sebelum menjalankan.

Tambahkan kredensial cloud

Gunakan API key uji yang dibatasi untuk instance dan firewall.

Buat node

Pilih region dan plan yang didukung. Plan lebih besar mendukung stack lengkap.

Tinjau detail protokol

Lihat port dan link SS, Hysteria2, VLESS-Reality, dan Trojan.

Gunakan port lokal

Halaman overview menampilkan port Mixed, HTTP, dan SOCKS langsung.

Impor ke tempat lain

Salin link ke sing-box, Clash, v2rayN, Hiddify, atau client kompatibel.

Format endpoint lokal

Format ini ditampilkan oleh panel port overview desktop.

Mixed

Salin sebagai host dan port.

127.0.0.1:7890

HTTP

Gunakan untuk app yang membutuhkan URL proxy HTTP.

http://127.0.0.1:7891

SOCKS

Gunakan untuk tool yang menerima endpoint SOCKS5.

socks5://127.0.0.1:7892

Panduan kredensial provider

Gunakan token uji khusus. Hindari memakai ulang token pribadi serbaguna.

Vultr

Gunakan key yang dapat membuat instance, mengelola firewall, membaca metadata, dan menghapus node uji.

DigitalOcean

Gunakan token yang dapat membuat droplets dan SSH keys. Recovery memakai SSH key terkelola.

SSH host

Gunakan host milik Anda atau yang diizinkan. Deploy memerlukan privilege layanan dan firewall.

Kegagalan umum

Sebagian besar masalah preview masuk kategori ini.

Provider menolak permintaan

Periksa scope token, saldo akun, region, dan quota.

SSH tidak pernah siap

Tunggu cloud-init, pastikan firewall dan dukungan SSH.

Port lokal tidak terhubung

Pastikan kernel, status proxy/TUN, dan link node sehat.
