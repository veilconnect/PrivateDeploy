Beta / Preview

PrivateDeploy

Alat untuk memasang node proxy pada VPS milik Anda. Buat server, deploy beberapa protokol, dan lihat port lokal Mixed, HTTP, SOCKS langsung di desktop.

Unduh preview Mulai cepat Model keamanan

Dirancang untuk mengelola node sendiri.

PrivateDeploy bukan hanya klien proxy lokal. Ia membuat VPS, memasang kredensial, menyinkronkan link protokol, dan menampilkan port lokal.

01

Provisioning cloud

Buat dan hapus node melalui provider yang didukung atau host yang bisa diakses SSH.

02

Deploy multi-protokol

Deploy Shadowsocks, Hysteria2, VLESS-Reality, dan Trojan jika paket mendukung.

03

Port terlihat jelas

Mixed, HTTP, dan SOCKS terlihat di halaman overview dan bisa langsung disalin.

04

Desktop, mobile, API

Gunakan aplikasi desktop Wails, klien Android, atau Go HTTP API.

Dashboard yang jelas untuk operasi.

Aplikasi desktop menampilkan traffic, mode, dan port lokal di tampilan pertama; panel cloud menyimpan port dan link setiap node.
