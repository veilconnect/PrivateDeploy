Beta / Preview

PrivateDeploy

अपने VPS पर प्रॉक्सी नोड तैनात करने का टूल। सर्वर बनाएं, कई प्रोटोकॉल तैनात करें और Mixed, HTTP, SOCKS लोकल पोर्ट सीधे देखें।

Preview डाउनलोड करें क्विक स्टार्ट सुरक्षा मॉडल

अपने नोड खुद चलाने के लिए बनाया गया।

PrivateDeploy केवल लोकल प्रॉक्सी क्लाइंट नहीं है। यह VPS बनाता है, credentials तैनात करता है, protocol links sync करता है और local ports दिखाता है।

01

Cloud provisioning

समर्थित providers या SSH hosts पर nodes बनाएं और हटाएं।

02

Multi-protocol deploy

Plan समर्थित होने पर Shadowsocks, Hysteria2, VLESS-Reality और Trojan तैनात करें।

03

Visible local ports

Mixed, HTTP और SOCKS ports overview page पर दिखते हैं और copy किए जा सकते हैं।

04

Desktop, mobile, API

Wails desktop app, Android companion या Go HTTP API इस्तेमाल करें।

Operational clarity के लिए dashboard।

Desktop app पहली screen पर traffic, mode और local ports दिखाता है; cloud panel हर node के ports और links रखता है।
